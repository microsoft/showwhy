# Components

This package contains generic reusable components to be used within te ShowWhy webapp.

## Folder structure

- src
  - ActionButtons
  - Callout
  - CardComponent
  - CausalEffectsArrow
  - CausalQuestion
  - DegreeDropdown
  - DetailsList
  - EffectScatterplot

  - PivotScatterplot
